# 🩰 Ânima Virtus

> Uma página institucional, para promover e dar visibilidade as aulas oferecidas.

## 🚀 Sobre o Projeto
O **Ânima Virtus** foi criado com a função de dar vitrine ao corpo docente e principalmente as aulas de ballet, também permite
a aproximação do potêncial aluno, via formulário ou links que direcionam a algum canal de comunicação.

### 🛠 Tecnologias
As principais ferramentas usadas na construção do projeto:
- [Next.js](https://nextjs.org)
- [Tailwind CSS](https://tailwindcss.com/)

## ✨ Funcionalidades
- [x] Contato por e-mail, via formulário.
- [x] Redirecionamento a canais de comunicação.

## 📦 Como rodar (site estático)

### Pré-requisitos
Nenhum — basta um navegador moderno.

### Passo a passo
```bash
1 Clone o repositório:
git clone https://github.com/brenotoyo/AnimaVirtus

2. Instale dependências
npm install
   ou
yarn install

# 3. Rode em modo de desenvolvimento
npm run dev     # Next.js
